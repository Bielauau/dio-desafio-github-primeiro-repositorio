### Topico: Git e Github

### SHA1

A sigla SHA significa Secure Hash Algorithm (Allgoritimo de Hash Seguro), é um conjunto de funções hash criptográficas projetadas pela NSA (Agência de Segurança Nacional dos EUA).
A encriptação gera conjuntos de characteres identificador de 40 dígitos .
O comando para utilização é: openssl sha1 

### Comandos

ls - lista os arquivos da pasta 

git status - mostra o status do commit

mv - para se mover entre diretórios 

git add - para adicionar/updata de um novo arquivo 

git add * - adiciona tudo de novo no arquivo 

git commit -m - para criar um commit 

git config —list - traz todas as configurações do git 

git config —global —unset - para resetar alguma configuração 

git config —global - para adicionar alguma configuração 

git remote add origin - para adicionar lista remota ao GitHub

git remote -v - mostra as listas remotas 

git push origin master - para enviar arquivos do repositório local para o remoto
